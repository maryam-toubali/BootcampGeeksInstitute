
🔁 What Are Loops?
Loops let you repeat a block of code multiple times instead of writing it again and again.

1️⃣ For Loops (for item in sequence)
Used to repeat actions a specific number of times or go through each item in a sequence (like a list, string, or tuple).

2️⃣ Ranges range(start, end)
The range() function creates a list of numbers. The end value is not included.

3️⃣ While Loops (while condition)
Repeat as long as a condition is True.

⚠ Using a Flag
You can control the loop with a variable (called a flag)
