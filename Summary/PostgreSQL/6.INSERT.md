| Form                          | Example                                                          | What It Does                             | When to Use It                                |
| ----------------------------- | ---------------------------------------------------------------- | ---------------------------------------- | --------------------------------------------- |
| Insert specific columns       | `INSERT INTO users (name, age) VALUES ('Sara', 28);`             | Adds a row with specified column values  | When adding new data manually or via a script |
| Insert all columns (in order) | `INSERT INTO users VALUES (1, 'Sara', 28);`                      | Adds a full row (must match all columns) | Risky unless you control column order         |
| Insert multiple rows          | `INSERT INTO users (name, age) VALUES ('Tom', 30), ('Ali', 25);` | Adds multiple rows at once               | Useful for bulk data import                   |
| Insert using DEFAULT          | `INSERT INTO users (name) VALUES ('Maya');`                      | Uses default values for missing columns  | If defaults are defined (e.g., created\_at)   |
