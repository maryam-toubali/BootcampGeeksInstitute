| Form                       | Example                                                         | What It Does                                      | When to Use It                               |
| -------------------------- | --------------------------------------------------------------- | ------------------------------------------------- | -------------------------------------------- |
| Update a single column     | `UPDATE users SET age = 30 WHERE name = 'Ali';`                 | Changes the age of one specific user              | Modify one or more fields for a specific row |
| Update multiple columns    | `UPDATE users SET name = 'Sara', age = 29 WHERE id = 3;`        | Changes more than one field in a row              | Useful in form submissions or batch updates  |
| Update all rows (⚠️ risky) | `UPDATE users SET active = true;`                               | Sets every row’s `active` column to `true`        | Be extremely cautious — use only when needed |
| Conditional update         | `UPDATE users SET age = age + 1 WHERE birthday = current_date;` | Dynamic logic (e.g., birthday-based age increase) | Good for scheduled updates                   |
