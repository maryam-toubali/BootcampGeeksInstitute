| Form                      | Example                                  | What It Does                    | When to Use It                                |
| ------------------------- | ---------------------------------------- | ------------------------------- | --------------------------------------------- |
| Select all columns        | `SELECT * FROM users;`                   | Gets all columns from all rows  | Quick check of full table                     |
| Select specific columns   | `SELECT name, age FROM users;`           | Returns only selected columns   | Efficient queries — only what you need        |
| With filtering (`WHERE`)  | `SELECT * FROM users WHERE age > 25;`    | Filters rows based on condition | Data analysis or specific reports             |
| With sorting (`ORDER BY`) | `SELECT * FROM users ORDER BY age DESC;` | Sorts results by column         | Rankings, reports, user-friendly data views   |
| With limit                | `SELECT * FROM users LIMIT 5;`           | Limits number of rows shown     | For previews, testing, and performance tuning |
| With aliases              | `SELECT name AS username FROM users;`    | Renames column in result        | For clean output or working with joins        |
