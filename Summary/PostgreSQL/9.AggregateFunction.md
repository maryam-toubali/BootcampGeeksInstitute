| 🔹 Function     | 🧩 Syntax Example                                                                | 🎯 What It Does                              | 💡 When to Use It                                          |
| --------------- | -------------------------------------------------------------------------------- | -------------------------------------------- | ---------------------------------------------------------- |
| `COUNT(*)`      | `SELECT COUNT(*) FROM actors;`                                                   | Counts total rows                            | To count all rows in a table, with or without conditions   |
| `COUNT(column)` | `SELECT COUNT(age) FROM actors;`                                                 | Counts non-null values in a column           | When you want to ignore NULLs                              |
| `SUM(column)`   | `SELECT SUM(number_oscars) FROM actors;`                                         | Adds up all values in a numeric column       | Total price, income, quantity, etc.                        |
| `AVG(column)`   | `SELECT AVG(number_oscars) FROM actors;`                                         | Returns the average of a column              | When calculating mean values like average salary or rating |
| `MIN(column)`   | `SELECT MIN(age) FROM actors;`                                                   | Finds the smallest (earliest/lowest) value   | Oldest date, lowest score, minimum price                   |
| `MAX(column)`   | `SELECT MAX(age) FROM actors;`                                                   | Finds the largest (latest/highest) value     | Newest date, highest salary, max rating                    |
| `GROUP BY`      | `SELECT last_name, COUNT(*) FROM actors GROUP BY last_name;`                     | Groups rows and applies aggregates per group | Use with aggregates to break down results by category      |
| `HAVING`        | `SELECT last_name, COUNT(*) FROM actors GROUP BY last_name HAVING COUNT(*) > 1;` | Filters group results                        | Like `WHERE`, but for grouped data (after `GROUP BY`)      |
