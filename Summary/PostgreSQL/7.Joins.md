### 🔹 Joins (with ON: like where)

| **Join Type**      | **What It Does**         | **Syntax**                                                |
| ------------------ | ------------------------ | --------------------------------------------------------- |
|`INNER JOIN / JOIN` | Returns matched rows     | `SELECT * FROM t1 INNER JOIN t2 ON t1.col = t2.col;`      |
| `LEFT JOIN`        | All from left + matches  | `SELECT * FROM t1 LEFT JOIN t2 ON t1.col = t2.col;`       |
| `RIGHT JOIN`       | All from right + matches | `SELECT * FROM t1 RIGHT JOIN t2 ON t1.col = t2.col;`      |
|`OUTER JOIN / UNION`| All from both sides      | `SELECT * FROM t1 OUTER JOIN t2 ON t1.col = t2.col;`      |
